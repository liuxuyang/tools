## Tutorial

#### 功能介绍
&#8194;&#8194;&#8194;&#8194;主要功能为 myos camera5.0 performance测试
#### 使用方法
* 打开待测手机开发者模式，使用usb链接PC，确认adb已连接（adb devices）
* 确保待测手机处于解锁状态
* 确认待测手机与config.json文件相匹配
* 命令行运行：project/path/kpi_test.sh -h 查看使用帮助
